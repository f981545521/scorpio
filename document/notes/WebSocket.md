## 如何使用WebSocket

1. Maven
```
        <dependency>
            <groupId>org.springframework</groupId>
            <artifactId>spring-websocket</artifactId>
        </dependency>
```
2. 配置:cn.acyou.scorpio.ws.MessageWebSocket.java

3. 配置：WebSocketConfig.java

4. 测试
- [在线测试地址1](http://coolaf.com/tool/chattest)
